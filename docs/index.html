MyBoard

Google Tasks와 Calendar를 한 곳에서 — 앱과 홈 화면 위젯으로.

소개

MyBoard는 개인이 개발한 무료 Android 앱으로, 사용자의 Google Tasks(할 일)와 Google Calendar(일정)를 하나의 화면과 홈 화면 위젯에서 통합 관리할 수 있게 해줍니다.

주요 기능

  • 할 일 목록 조회 및 추가·수정·완료·삭제
  • 캘린더 일정 조회 및 생성·수정·삭제
  • 홈 화면 위젯에서 할 일·일정 바로 확인 및 빠른 추가
  • 커버 화면·태블릿 등 다양한 화면 크기 지원

Google 데이터 사용 방식

앱은 Google 로그인 후 사용자 본인의 Google Tasks 및 Calendar 데이터에만 접근합니다. 모든 데이터는 Google API를 통해 직접 조회되며 사용자 기기 내에서만 처리됩니다. 어떤 데이터도 외부 서버나 제3자에게 전송·저장·판매되지 않습니다.
  • Google Tasks — 할 일 목록 표시 및 관리
  • Google Calendar — 일정 표시 및 관리

앱의 Google 사용자 데이터 사용은 Google API Services User Data Policy를 준수합니다.

MyBoard is a free, personally developed Android app that displays and manages the user's own Google Tasks and Calendar in a single app and home-screen widgets. All data is accessed via Google APIs and processed only on the user's device; nothing is sent to external servers or third parties.
